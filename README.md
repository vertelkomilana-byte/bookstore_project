Проект: Аналитика книжного магазина

Технологии:
- SQL (SQLite)
- Python

Описание:

Учебный проект для отработки SQL-запросов и работы Python с базой данных.

Структура проекта

schema.sql — создание таблиц базы данных
data.sql — наполнение базы тестовыми данными
queries.sql — SQL-запросы для анализа данных
init_db.py — инициализация базы данных (создание и заполнение)
main.py — запуск проекта и выполнение запросов
database.db — файл базы данных SQLite.

Логика работы проекта

1. Создаётся база данных SQLite на основе схемы из schema.sql
2. Таблицы наполняются тестовыми данными из data.sql
3. SQL-запросы для анализа данных хранятся в файле queries.sql
4. Скрипт main.py подключается к базе данных и выполняет SQL-запросы
5. Результаты запросов выводятся в консоль
